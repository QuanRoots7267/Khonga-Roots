# Khonga Roots Website

A modern, responsive website celebrating cultural heritage and authentic traditions.

## Project Structure

```
khonga-roots/
├── index.html          # Main homepage
├── style.css           # Styling and responsive design
├── script.js           # Interactive functionality
└── README.md          # Project documentation
```

## Features

- ✨ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- 🎯 **Navigation** - Smooth scrolling navigation with mobile hamburger menu
- 📱 **Mobile-Friendly** - Optimized layout for all screen sizes
- 💬 **Contact Form** - Functional contact form for inquiries
- 🎨 **Modern UI** - Clean and professional design with smooth animations
- ⚡ **Fast Loading** - Lightweight and optimized performance

## Sections

1. **Hero Section** - Eye-catching banner with call-to-action
2. **About Us** - Information about Khonga Roots with feature highlights
3. **Services** - Overview of services offered
4. **Contact** - Contact form for inquiries
5. **Footer** - Social links and copyright information

## Color Scheme

- Primary Color: `#2d5016` (Forest Green)
- Secondary Color: `#8b7355` (Brown)
- Accent Color: `#d4af37` (Gold)
- Light Background: `#f5f5f5`

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the sections using the menu
3. Fill out the contact form to get in touch

## Customization

- Replace the placeholder text and information with your actual content
- Update social links in the footer
- Add your own images to an `assets/` folder
- Modify colors in the `:root` CSS variables
- Add more sections as needed by duplicating existing section HTML and CSS

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers
